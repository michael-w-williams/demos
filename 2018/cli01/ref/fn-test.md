# `fn test`

```yaml
$ fn test

DEVELOPMENT COMMANDS
  fn test -   Run functions test
              
USAGE
  fn [global options] test [command options] [function-subdirectory] 
    
DESCRIPTION
  This commmand runs fn tests if present.
    
COMMAND OPTIONS
  --remote value                 Run tests against a remote fn server
  --all app.yaml                 If in root directory containing app.yaml, this will deploy all functions
  --working-dir value, -w value  Specify the working directory to test a function, must be the full path.
```
