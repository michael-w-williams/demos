# `fn create`

```yaml
$ fn create

MANAGEMENT COMMAND
  fn create -   Create a new object
                
USAGE
  fn create [global options] <object-type> [command options]
    
DESCRIPTION
  This command creates a new object ('app', 'context' or 'route').
    
SUBCOMMANDS
  app, apps, a               Create a new application
  context, ctx               Create a new context
  function, func, f, fn      Create a function in an application
  route, routes, r           Create a route in an application
  trigger, trig, t, tr       Create a new trigger
  help, h                    Shows a list of commands or help for one command
                         
COMMAND OPTIONS
  --help, -h  show help

FURTHER HELP:  See 'fn create <subcommand> --help' for more information about a command.
```
