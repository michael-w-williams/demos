# `fn create function`

```yaml
$ fn create function


  fn create function - Create a function in an application
    
USAGE
  fn create function [global options] <app> <fnname> <image> [command options]
    
COMMAND OPTIONS
  --memory value, -m value  memory in MiB (default: 0)
  --config value, -c value  fn configuration
  --format value, -f value  hot container IO format - default or http
  --timeout value           fn timeout (eg. 30) (default: 0)
  --idle-timeout value      fn idle timeout (eg. 30) (default: 0)
  --annotation value        fn annotation (can be specified multiple times)
  
```
